3D Mesh Layout Classification Research

Research Assistant Internship at Yuan Ze University, Taiwan (September 2024 â€“ January 2025)

Research Overview

During my internship at Yuan Ze University, I contributed to a research project focused on classifying the quality of 3D object layouts using machine learning and deep learning approaches. The research addressed the challenge of ensuring quality in 3D object models for applications in computer vision, mixed reality, and graphics.

Our study evaluated the performance of various classification models on six diverse datasets, providing insights into optimizing classification pipelines for 3D models.

Datasets

We worked with six diverse datasets containing 3D object models with layout quality annotations:

Dataset Samples Description
3D-FUTURE 16,564 Layout levels and confidence annotated by two persons
ABO 7,952 Rich annotations including layout and inside levels with confidence scores
HermanMiller 1,589 Specific layout and inside level annotations with confidence
IKEA 477 Layout and inside levels per object, linking to folders
Pix3D 395 Detailed layout levels annotated by four people
ShapeNetCore 51,507 Extensive layout annotations from up to seven annotators

My responsibilities included data preprocessing, validation, and ensuring consistency between annotations and object files.

Methodology

We implemented and evaluated five different classification models:

Traditional Machine Learning

SVM: Support Vector Machine for finding optimal hyperplanes to separate classes

Random Forest: Ensemble method combining multiple decision trees

XGBoost: Advanced gradient-boosting algorithm optimizing predictive performance

Deep Learning Architectures

PointNet: Processes point clouds directly using shared MLPs and max-pooling

PointNet++: Extends PointNet with hierarchical feature learning

MeshNet: Processes meshes using their geometric and structural properties

I was primarily responsible for implementing the data preprocessing pipeline, feature extraction, and model evaluation components. This included:

  • Developing robust data validation procedures to ensure consistency between annotations and 3D models
  • Implementing coordinate standardization and scaling to normalize 3D objects
  • Extracting geometric features from 3D meshes for traditional machine learning models
  • Assisting with the implementation and optimization of deep learning architectures

Key Findings

Model Performance

MeshNet emerged as the top performer across most datasets, achieving up to 96% accuracy on the Pix3D dataset. PointNet++ consistently outperformed PointNet, demonstrating the importance of hierarchical feature learning for 3D object classification.

Other significant findings included:

  • Deep learning models (PointNet, PointNet++, MeshNet) consistently outperformed traditional machine learning approaches on complex 3D data
  • MeshNet's ability to process both vertices and normals provided a significant advantage in capturing fine-grained geometric details
  • Data augmentation techniques significantly improved model generalization, particularly for MeshNet
  • Early stopping and dropout layers effectively controlled overfitting in deep learning models
